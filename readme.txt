== Description ==

Allows you to select styling for the most popular IDX providers who integrate with WordPress. This plugin only works with Genesis child themes created by Agent Evolution.


== Installation ==

1. Upload the entire `ae-idx-styler` folder to the `/wp-content/plugins/` directory
1. DO NOT change the name of the `ae-idx-styler` folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to the `Appearance > IDX Styler` menu
1. Select an IDX provider.
1. Select the Agent Evolution theme you are using.
